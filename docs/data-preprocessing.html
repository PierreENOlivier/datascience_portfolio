Some of the latin name abbreviations are associated with the wrong organism. The abbreviations follow international fisheries standards (e.g.Â GAD_MOR for âGADUS MORHUAâ). We can reconstruct the abbreviations from the organismsâ names.


# "(^[A-Za-z]{1,3})"

abbr2 <- species_list$TROPHOSPECIES %>% str_extract(., "(?<=_)([A-Za-z]{1,3})")
species_list$ABBREVIATION <- paste(abbr1, ifelse(is.na(abbr2), "IND", abbr2), sep="_" )

4.4.2 Data transformation

The data was provided as a long pairwise list which is limiting for the purpose of the analysis. We can convert the list to a squared binary matrix where the intersections of rows and columns contain the information for an interaction.

In other words, â0sâ means that the interaction is absent (or has yet to be observed in nature); â1sâ indicates the presence of an interaction between the organisms listed in the columns and rows interacting.
